# Godot Folder

This folder contains the Godot 4 project for Cuticle Capital.

## Files
- `project.godot`: project settings and run scene.
- `scenes/Main.tscn`: main scene.
- `scripts/main.gd`: economy loop, upgrades, queue demand, missions, assistant automation, telemetry hooks.
- `data/economy.json`: data-driven economy values.
- `assets/`: imported art/audio assets grouped by type.

## Web Build Output
- `build/web/`: exported HTML5 artifacts (`index.html`, `.wasm`, `.js`, `.pck`).

## Current Imported Art
- `assets/fonts/kenney/`: UI font used by runtime theme.
- `assets/ui/kenney/`: button skins and icons used by runtime controls.
- `assets/sprites/kenney/`: decorative customer sprites in the HUD.
- `assets/tilesets/kenney/`: floor tile texture used as background layer.
