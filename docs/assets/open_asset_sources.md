# Free/Open Asset Sources

Use these sources for production-safe free assets. Prefer CC0 first.

## Recommended Sources
- Kenney (CC0-focused packs): https://kenney.nl/assets
- OpenGameArt (license varies per pack): https://opengameart.org/
- Game-icons (CC BY 3.0 default unless otherwise noted): https://game-icons.net/about.html
- Freesound (filter by license before use): https://freesound.org/help/faq/

## License Checks (Official References)
- Kenney support/licensing info: https://kenney.nl/support
- OpenGameArt licensing FAQ: https://opengameart.org/content/faq
- Game-icons licensing details: https://game-icons.net/license.html
- Freesound license overview: https://freesound.org/help/faq/#licensing

## Import Rules for This Repo
1. Prefer CC0 packs for core UI + tiles + sprites + SFX.
2. If CC-BY assets are used, add required credits in `docs/assets/ATTRIBUTION.md`.
3. Record source URL, author, and license for every imported pack.
4. Keep imported runtime files under `godot/assets/` grouped by type.
